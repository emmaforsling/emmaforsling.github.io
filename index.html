SumoBot Wars

SumoBot Wars is a game created in Unreal Engine 4. A single player mode exists, in which the player fights against an AI developed utilizing a reactive behavior tree. A multiplayer mode also exists, in which two players can compete against each other. The inspiration for this game comes from the sport SumoBot in which two robots try to push each other one out of an arena.

Nominated in Liu Game Awards 2015
